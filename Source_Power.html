# [SOURce<HW>]:POWer:MANual
value: float = driver.source.power.get_manual()

Sets the level for the subsequent sweep step if method RsSmbv.source.sweep.power.mode.value() . Use a separate command for each sweep step.

Returns:

manual: float You can select any level within the setting range, where: STARt is set with method RsSmbv.source.power.start() STOP is set with method RsSmbv.source.power.stop() OFFSet is set with method RsSmbv.source.power.level.immediate.offset() Range: (STARt + OFFSet) to (STOP + OFFSet) , Unit: dBm

Selects the operating mode of the instrument to set the output level.

Returns:

mode: CW | FIXed | SWEep CW | FIXed Operates at a constant level. CW and FIXed are synonyms. To set the output level value, use the command method RsSmbv.source.power.level.immediate.amplitude() .
